How to test this:

  1. Click "Category 1". The page reloads, and that link turns Blue (Active).
  2. Click "Marketing Link". The URL changes, but the script knows to ignore `utm_source`, so "Demo Home" stays Blue.
  3. The "Examples Folder" link should have an Orange underline (Ancestor) because this file is inside that folder.

Note: For the most accurate testing, run this via a local server (like `python3 -m http.server` or `npx serve --symlinks` or VS Code Live Server) rather than opening the file directly, as file:// protocols can behave strangely with `window.location.origin`.
